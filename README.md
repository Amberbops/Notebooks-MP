# Notebooks-MP

A minor project for managing and organizing notebooks.

## Overview

Notebooks-MP is a project designed to help users efficiently manage, organize, and maintain their notebook collections. Whether you're tracking personal notes, study materials, or project documentation, this tool provides a streamlined solution.

## Features

- 📓 Create and manage multiple notebooks
- 🏷️ Organize content with tags and categories
- 🔍 Search and filter functionality
- 📝 Rich text editing support
- 💾 Local storage and persistence

## Getting Started

### Prerequisites

- Python 3.8 or higher
- Git

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Amberbops/Notebooks-MP.git
cd Notebooks-MP
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
python main.py
```

## Usage

### Basic Operations

- **Create a Notebook**: Start a new notebook with a custom title
- **Add Notes**: Add content to your notebooks with timestamps
- **Organize**: Use tags and categories to keep notes organized
- **Search**: Find notes quickly using the search functionality
